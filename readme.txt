This plugin connects your WooCommerce store with your Zendesk account.
== Description ==
***Easy connect WooCommerce with Zendesk, directly create new Zendesk end users and organizations from new customers, let customers post new tickets within WooCommerce and connect your Zendesk support webportal to your store.***

The _WooCommerce Zendesk Connect plugin_ allows shop owners to connect their WooCommerce shop with a Zendesk account. Save time and import your customers easily into your Zendesk.

For more information and a complete list of features, check out [our website](https://wpfortune.com/shop/plugins/woocommerce-zendesk-connect/).

This is just a first release and we've a lot of features we want to add in next releases.

This plugin is compatible with Wordpress 4.3 and WooCommerce 2.4.x.

**Features**

* Create Zendesk end users and organizations from your WooCommerce customers
* Add support ticket form on WooCommerce My Account page
* Localisation: English & Dutch

== Installation ==

1. Install WooCommerce Zendesk Connect either via the Wordpress.org plugin directory or by uploading the files to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to WooCommerce --> Zendesk Connect
4. Fill in your Zendesk account information in tab "Authentication"
5. Configure plugin and save settings.
